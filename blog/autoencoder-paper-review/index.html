[논문 리뷰] 딥러닝 기초 - Auto-Encoder

Hinton의 오토인코더 논문을 바탕으로 차원 축소, Manifold Learning, PCA와의 비교, 그리고 PyTorch 구현까지 정리합니다.

AI Deep Learning 논문리뷰 AutoEncoder

Series: 딥러닝 기초

  1. 1. [논문 리뷰] 딥러닝 기초 - Auto-Encoder

개요

Geoffrey Hinton, Ruslan Salakhutdinov (2006) “Reducing the Dimensionality of Data with Neural Networks”

딥러닝의 대부 Geoffrey Hinton이 제안한 오토인코더는 차원 축소 및 특징 추출을 위한 신경망 구조입니다. PCA의 선형적 한계를 뛰어넘어 비선형 Manifold를 학습할 수 있다는 점에서 큰 의미를 가집니다.

오토인코더 구조

오토인코더는 세 가지 핵심 구성요소로 이루어집니다:

인코더 (Encoder)

  • 고차원 입력을 저차원 은닉층(Latent Space)으로 압축
  • 차원이 높을수록 모델 복잡성 증가

코드 / Latent Space

  • 인코더-디코더 연결점의 은닉층
  • 입력 데이터의 압축된 형태로, 중요한 특징만 포함

디코더 (Decoder)

  • 저차원 은닉층을 고차원 출력으로 복원
  • 완벽하지 않지만 최대한 원본 데이터 복원 시도

등장 배경: 왜 차원 축소가 필요한가

차원의 저주 (Curse of Dimensionality)

차원이 증가하면 데이터의 개수가 차원 크기보다 작아져 학습 효율이 급격히 저하됩니다.

문제점:

  • 데이터 밀도 희박화
  • 필요한 샘플 수 기하급수적 증가
  • KNN, SVM, GMM 등의 성능 악화

해결 방법:

  • PCA 같은 차원 축소 기법
  • 훈련 샘플 크기 증가
  • 오토인코더를 통한 비선형 차원 축소

Manifold Learning

Manifold란 데이터가 존재하는 부분공간으로, 데이터를 최적으로 설명하는 저차원 구조입니다.

Manifold Learning의 목적

Data Compression

  • 코드층으로 압축 후 원본으로 복원
  • 고차원 → 저차원 변환

Data Visualization

  • t-SNE 활용으로 비슷한 구조 정리

Curse of Dimensionality 개선

  • Manifold 발견으로 데이터 밀도 증가
  • 저차원 표현에서 높은 밀도 유지

Feature Discovery

  • 유의미한 특징 자동 추출
  • MNIST의 크기, 회전, 변형, 두께 등 학습

핵심: Manifold 위의 점 선택과 Euclidean 공간의 중점 선택은 다릅니다. 의미 있는 데이터 생성을 위해서는 Manifold 위에서 보간이 필수적입니다.

PCA (주성분 분석)

개념

고차원 데이터를 저차원으로 변환하는 선형 기법입니다.

수학적 원리

분산이 최대인 방향이 주성분입니다:

분산=wTCw\text{분산} = w^T C w분산=wTCw

여기서 CCC는 공분산 행렬입니다. 라그랑주 승수법으로 제약 최적화하면:

Cw=λwCw = \lambda wCw=λw

이는 고유값 분해(eigendecomposition)로 귀결됩니다. 최대 분산 방향이 고유벡터이고, λ\lambdaλ는 분산의 크기입니다.

PCA의 한계

  • 항상 선형 결과만 도출
  • 비선형 구조(곡면 등)를 포착할 수 없음

다층 퍼셉트론에서의 착안

L1 = W1 * x^T
L2 = W2 * L1 = W2 * W1 * x^T

PCA에 비선형성을 추가할 수 있을까? → 오토인코더의 탄생

오토인코더 학습: Pretraining

문제: 초기값 설정의 중요성

당시 깊은 네트워크 학습 실패의 주요 원인은 부적절한 가중치 초기화였습니다.

제한된 볼츠만 머신 (RBM)

구조: Visible-Hidden 양층, 완전연결

훈련 과정:

  1. Forward: 입력 → 가중치 → 출력
  2. Backward: 출력 → 동일 가중치 → 입력 재생성
  3. 원본-재생성 입력 비교 후 가중치 업데이트

RBM을 층층이 쌓아 Deep Belief Network(DBN)을 구성하고, 이를 통해 가중치를 초기화했습니다.

현대적 초기화 방법

Xavier 초기화 (2010):

W∼N(0,2nin+nout)W \sim N\left(0, \frac{2}{n_{in} + n_{out}}\right)W∼N(0,nin​+nout​2​)
  • Sigmoid/Tanh 활성화 함수에 적합
  • 입출력 뉴런 수 기반 동적 조절

He 초기화 (2015):

W∼N(0,2(1+a2)⋅nin)W \sim N\left(0, \frac{2}{(1+a^2) \cdot n_{in}}\right)W∼N(0,(1+a2)⋅nin​2​)
  • ReLU 활성화 함수에 적합
  • 음수 부분 기울기 반영

Xavier/He 초기화의 등장으로 RBM 기반 사전학습은 거의 사용되지 않게 되었습니다.

스택 오토인코더 (Stacked Autoencoder)

층별로 학습한 뒤 전체 네트워크를 Fine-tuning하는 방식:

Input → 2000 → 1000 → 500 → 30 → 500 → 1000 → 2000 → Output

자가지도학습 (Self-supervised Learning):

  • 입력 데이터 자체가 타깃값
  • 레이블 없는 데이터에서도 효과적 학습
  • 재구성 과정을 통한 표현 학습

PCA vs Auto-Encoder 비교

Swiss Roll 실험

from sklearn.datasets import make_swiss_roll
n_samples = 3000
X, colors = make_swiss_roll(n_samples)

PCA:

from sklearn.decomposition import PCA
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

Auto-Encoder:

from keras.layers import Input, Dense
from keras.models import Model

input_img = Input(shape=(3,))
encoded = Dense(2, activation='relu')(input_img)
decoded = Dense(3, activation='sigmoid')(encoded)

autoencoder = Model(input_img, decoded)
encoder = Model(input_img, encoded)

autoencoder.compile(optimizer='adadelta', loss='binary_crossentropy')
autoencoder.fit(X, X, epochs=10, batch_size=256)
X_auto = encoder.predict(X)

결론: 오토인코더는 PCA와 달리 선형 평면이 아닌 곡면(Manifold)을 학습할 수 있습니다.

PyTorch 구현

모델 정의

import torch
from torch import nn

class Autoencoder(nn.Module):
    def __init__(self):
        super(Autoencoder, self).__init__()

        self.encoder = nn.Sequential(
            nn.Linear(28 * 28, 128),
            nn.ReLU(inplace=True),
            nn.Linear(128, 64),
            nn.ReLU(inplace=True),
            nn.Linear(64, 12),
            nn.ReLU(inplace=True),
            nn.Linear(12, 3),
        )

        self.decoder = nn.Sequential(
            nn.Linear(3, 12),
            nn.ReLU(inplace=True),
            nn.Linear(12, 64),
            nn.ReLU(inplace=True),
            nn.Linear(64, 128),
            nn.ReLU(inplace=True),
            nn.Linear(128, 28 * 28),
            nn.Sigmoid(),
        )

    def forward(self, x):
        x = self.encoder(x)
        x = self.decoder(x)
        return x

학습

from torch.utils.data import DataLoader
from torchvision import datasets, transforms

transform = transforms.Compose([transforms.ToTensor()])
train_dataset = datasets.MNIST(
    root="./data", train=True, transform=transform, download=True
)
train_loader = DataLoader(train_dataset, batch_size=128, shuffle=True)

model = Autoencoder()
criterion = nn.MSELoss()
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3, weight_decay=1e-5)

for epoch in range(20):
    for img, _ in train_loader:
        img = img.view(img.size(0), -1)
        output = model(img)
        loss = criterion(output, img)

        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

    print(f"epoch [{epoch+1}/20], loss:{loss.item():.4f}")

핵심 결론:

  1. 오토인코더는 PCA보다 비선형 Manifold 학습 가능
  2. 다양한 차원 축소 작업에서 우수한 성능 입증
  3. 레이블 없는 데이터에서 효과적인 특징 추출
  4. 이미지, 텍스트 등 다양한 도메인에 적용 가능
